# Parch Linux dwm

parch linux dwm is a release of parch linux with dynamic window manager.

this release contains a patched version of dwm with kitty and dmenu.


### Dwm Keys


**opens terminal**
``super + return``

**opens file manager**
``super + shift + return ``

**opens dmenu**
``super + p``

**opens logout window**
``super + shift + q``

**opens dm_hub for screenshot or etc**
``super + shift + x``

**kill active window**
``super + shift + c``

**cycle between the workspaces**
``super + 1-9``

**move active window to another workspace**
`` super + shift + 1-9 ``
